
# 🎬 Movie Night Planner

## 📌 Description

This project is a simple JavaScript-based Movie Night Assistant.
It allows users to log in, select a movie package, calculate total cost, and split the bill among a group.

---

## 🚀 Features

* 🔐 User login system (admin / guest)
* 🍿 Movie selection (standard, 3D, IMAX)
* 🎟 Ticket cost calculation
* 🥤 Optional snacks selection
* 💸 Discount for more than 3 tickets
* 👥 Bill splitting with service fee

---

## 🛠 Technologies Used

* JavaScript (prompt, alert)
* Basic programming concepts (if/else, switch, operators)

---

## ▶️ How to Run

1. Open a browser
2. Open Developer Console (Right click → Inspect → Console)
3. Copy and paste the `index.js` code
4. Press Enter
5. Follow the prompts

---

## 📊 Example

* Select movie type
* Enter number of tickets
* Choose snacks and service fee
* View total cost and split per person
